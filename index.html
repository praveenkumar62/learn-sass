SASS Installation

Method - 1 [Using Prepros Software]

Install the software called Prepros, open the project folder and create folder for CSS, then select SCSS file from SCSS folder and process, this tool automatically processes the SCSS file into CSS and keeps in css folder

Method - 2 [Using NPM - sass (installing globally only)]

Open code editor and prepare SCSS files in folder along with creation of new folder for CSS. Open terminal, install sass by "npm install -g sass" [Installing globally helps SCSS to work in local].
Type "sass --watch <SCSS folder or SCSS file in root>/<SCSS folder or SCSS file> <CSS folder in root>/<next CSS folder or CSS file>"
EX: "sass --watch scss/styles.scss css/styles.css"
Terminal will continue to watch for SCSS changes.

Method - 3 [VS Code extension - live sass compiler]

Open VS Code editor, install plugin - live sass compiler. close and navigate to editor settings [bottom left corner]. Search and enter into live sass compiler settings. Click on Edit in settings.json 

    
        "liveSassCompile.settings.generateMap":false, // not required to generate .map.css
        "liveSassCompile.settings.formats": [
            {
                "format": "expanded", // options - expanded (or) compressed
                "extensionName": ".css", // options - min.css or .css
                "savePath": "/css" // folder directory for storing css
            }
        ]
Need to change this setting only once, when "live sass compiler" is installed.

Method - 4 [Using NPM - node sass (for JS libraries & frameworks)]

Step 1: In project folder, perform npm init in terminal and setup the node modules

Step 2: Install "npm istall node-sass", then create folder for SCSS and CSS

Step 3: Open package.json and add/replace the following 

        
            "scripts": {
                "sass": "node-sass -w <root folder -> SCSS folder or SCSS folder>/ -o <root folder -> CSS folder or CSS folder> --recursive"
            }

Step 4: Now run "npm run sass" in the terminal

Step 5: Click save to compile SASS. [Even when autosave turned on]
